Safe analysis workspace

Review the mess first. Decide later.

Map storage, mark favorites, hide noise, inspect file contents, and find duplicate or stale file candidates without modifying the scanned folder. The only writes are your local review notes and card states.

Local notes only Duplicate scan is read-only Protected paths stay flagged

Good starting points

Use a folder you actually own first, like Downloads, Desktop, Documents, or a project folder.

  • The map shows the largest folders quickly.
  • Review cards can be favorited, hidden, or marked reviewed.
  • The duplicate and stale-file views stay read-only.

What “write” means here

Only your local review state is saved into the app workspace. The app never deletes, moves, or edits the scanned files.
